## Getting Started

### Prerequisites

- Node.js 18.0.0 or higher
- npm or yarn

### Installation

```bash
git clone https://github.com/minpeter/domain-expiry-lookup.git
cd domain-expiry-lookup
npm install
```

### Development Server

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
